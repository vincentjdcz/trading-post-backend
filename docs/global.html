(async) signup(req, res) → {Promise.<void>}

Creates a new User document.
Parameters:
Name Type Description
req Object Express request object
Properties
Name Type Description
body Object Request body
params.userName string Desired userName of the user creating an account.
params.firstName string First name of the user creating an account.
params.lastName string Last name of the user creating an account.
params.email string Email address of the user creating an account.
params.password string Desired password of the user creating an account.
res Object Express response object
Source:
  • auth/controller.js, line 5
Throws:
  • Returns a 400 status code if an account with the desired userName already exists.
  • Returns a 400 status code if an account with the given email address already exists.
  • Returns a 500 status code if an error occurs while creating the new User.
Returns:
Returns a JSON response with a success or error message.
Type
Promise.<void>
Example
// Example POST request to create a user with provided userName, firstName, lastName, email and password
// Endpoint: GET /signup
const devURL = `http://localhost:3000/api/auth/signup`;        
const response = await fetch(devURL,
  {
    method: "POST",
    headers: { "Content-Type": "application/json" },
    body: JSON.stringify({
      userName,
      firstName,
      lastName,
      email,
      password,
    }),
  }
);
